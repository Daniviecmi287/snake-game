# 🐍 Snake.io — Classic Arcade Game

A modern take on the classic Snake game built with **pure Vanilla JavaScript** — no frameworks, no dependencies. Features a sleek dark-neon UI, smooth particle effects, progressive difficulty levels, and full mobile support.

![Snake Game Screenshot](https://raw.githubusercontent.com/Daniviecmi287/snake-game/main/screenshot.png)


---

## ✨ Features

- 🎮 **Keyboard & Mobile** — Arrow keys, WASD, on-screen buttons, and swipe gestures
- 🏆 **High Score** — Persisted via `localStorage` across sessions
- ⚡ **Progressive Difficulty** — Speed increases every 5 foods eaten
- 💥 **Particle Effects** — Burst animations on food collection
- 🌙 **Dark Neon UI** — Custom retro-futuristic design with CSS animations
- 📱 **Responsive** — Scales cleanly on all screen sizes
- ♿ **Accessible** — Keyboard-first controls, clear visual feedback

## 🚀 Getting Started

### Option 1 — Open directly
```bash
# Clone the repository
git clone https://github.com/Daniviecmi287/snake-game.git

# Navigate into the folder
cd snake-game

# Open in your browser (no build step needed)
open index.html       # macOS
start index.html      # Windows
xdg-open index.html   # Linux
